# 配置 SSH

> TIP：本文以 CentOS7 为例进行演示。

## 安装 Git

```bash
sudo yum install git -y
```

进行相关配置，以 `user.name` 和 `user.email` 为例：

```java
git config --global user.name "Your Name"
git config --global user.email "xxx@xxx.xx"
```

## 创建 SSH 密钥

```bash
ssh-keygen -t rsa
```

一路回车，生成的密钥文件在 `~/.ssh/id_rsa` 和 `~/.ssh/id_rsa.pub`。由于 `.ssh` 是隐藏文件夹，可以使用 `ls -al ~/.ssh` 查看。复制 `~/.ssh/id_rsa.pub`
文件内容，添加到 GitLab 或 GitHub 的 SSH 密钥中即可。
