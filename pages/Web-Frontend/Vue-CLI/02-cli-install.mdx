# Vue CLI 安装指南

## Node 版本要求

要使用 Vue CLI 4.x，您的系统需要安装 Node.js v8.9 或更高版本，推荐使用 v10 及以上版本。为了方便管理多个 Node 版本，您可以使用以下工具：

- **n**：简单易用的 Node 版本管理工具。
- **nvm**：流行的 Node 版本管理器，支持在同一台电脑中切换不同版本的 Node.js。（推荐）
- **nvm-windows**：nvm 的 Windows 版本，适用于在 Windows 系统上管理多个 Node 版本。

## 安装 Vue CLI

可以通过以下任一命令来全局安装 Vue CLI：

```bash
npm install -g @vue/cli
# 或者
yarn global add @vue/cli
```

安装完成后，您应该能够在命令行中使用 `vue` 命令。为确保安装成功，您可以在终端中简单运行 `vue`，查看是否展示了一份包含所有可用命令的帮助信息。



此外，您还可以通过以下命令检查 Vue CLI 的版本，确保安装正确：

```bash
vue --version
```

## 升级 Vue CLI

如果需要升级全局安装的 Vue CLI 包，可以使用以下命令：

```bash
npm update -g @vue/cli
# 或者
yarn global upgrade --latest @vue/cli
```

## 升级项目依赖

上述命令用于升级全局的 Vue CLI。如果您需要升级项目中的 Vue CLI 相关模块（这些模块通常以 @vue/cli-plugin- 或 vue-cli-plugin- 开头），可以在项目目录下运行 vue upgrade 命令：

```bash
vue upgrade [options] [plugin-name]
```

该命令支持以下选项：

- **`-t, --to <version>`**：将指定的插件升级到特定版本。
- **`-f, --from <version>`**：跳过本地版本检测，假设插件是从指定的版本升级上来。
- **`-r, --registry <url>`**：使用指定的 registry 地址安装依赖。
- **`--all`**：升级项目中所有的插件。
- **`--next`**：在检查插件新版本时，包括 alpha、beta 和 rc 版本在内。
- **`-h, --help`**：输出帮助信息。

通过这些选项，您可以灵活地管理和升级项目中的 Vue CLI 服务及其插件，确保您的开发环境保持最新和最佳状态。
