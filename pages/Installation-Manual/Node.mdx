拉取 Node 的 Docker 镜像，具体版本可以在上面的官方文档中选择：

```basn
docker pull node:14-alpine
```

验证环境中是否有正确的 Node.js 版本：

```basn
docker run node:14-alpine node -v
```

验证环境中是否有正确的 NPM 版本：

```basn
docker run node:14-alpine npm -v
```

如果想设置镜像源，则打开安装路径下的 settings.txt 在末尾添加如下配置：

```bash
node_mirror: https://npmmirror.com/mirrors/node/
npm_mirror: https://npmmirror.com/mirrors/npm/
```

---
> TIP：下面介绍 MacOS 和 Linux 安装方式。

使用 curl 或 wget 安装 NVM。以下是使用 curl 的命令：

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

或者使用 wget：

```bash
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

> TIP: 安装脚本会克隆 NVM 仓库到 `~/.nvm`，并将脚本添加到 shell 配置文件中 (`~/.bash_profile`, `~/.zshrc`, `~/.profile`, 或 `~/.bashrc`)。

为使 NVM 生效，需要重启终端或执行：

```bash
export NVM_DIR="$([ -z "${XDG_CONFIG_HOME-}" ] && printf %s "${HOME}/.nvm" || printf %s "${XDG_CONFIG_HOME}/nvm")"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
```

安装特定版本的 Node.js：

```bash
nvm install 18
```

设置默认 Node.js 版本：

```bash
nvm use 18
nvm alias default 18
```

pnpm config set disturl https://npmmirror.com/mirrors/node
